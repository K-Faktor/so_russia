# so_russia
SO Russia Promod

## Required files and folders to run the mod/server

    /maps/
    /promod/
    /duffman/
    /scripts/
    /server binary
    /config.cfg
    /mod.ff
    / .iwd files, depending how you split the assets

IWD files contain /images, /sounds and /weapons
